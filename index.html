Atalhos Essenciais do VS Code

Por: Marcelo Paludetto

Aumente sua produtividade com estes comandos fundamentais:

  • Alt + Z - Ativa quebra automática de linha.
  • Ctrl + B - Exibe ou oculta a barra lateral.
  • Ctrl + P - Pesquisa arquivos rapidamente.
  • Ctrl + Shift + K - Apaga a linha atual.
  • Ctrl + / - Comenta ou remove comentário.
